Motivos para utilizar o Bootstrap

Modularidade

O framework abrange a mior parte dos componentes utilizados em websites modernos.

Customizabilidade

utilizando os atalhos inline, é facil customizar os elementos das paginas.

Responsividade

com breakpoints de tamanho pré definidos e um grid bem definido, é trivial fazer paginas responsivas em qualquer dispositivo.

Facilidade no uso

Com uma documentação robusta, o bootstrap é de fácil utilização, onde basta selecionar um elemento pré criado e alterar apenas alguns parametros para ter o elemento perfeito para seu website.
